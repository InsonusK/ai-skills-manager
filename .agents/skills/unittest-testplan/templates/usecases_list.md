# Template of usecases list
When you need confirm list of usecases for unittes. Use [template below](#template) to show them for user confirmation.

## Template
```
- {Class}
    - {Method}
        - {Group}
            - WHEN_{condition}_THEN_{result}
                - {description}
                - {input}
                - {output}
                - {expected_result}
```
